# よく使うコマンド
- 開発サーバ: npm run dev
- ビルド: npm run build (tsc -b && vite build)
- テスト: npm test / npm run test:coverage / npm run test:ui
- Lint: npm run lint
- フォーマット: npm run format (checkは npm run format:check)
- チェック一括: npm run check (lint + format:check)
- プレビュー: npm run preview
- 環境: macOS + zsh。標準Unix系ユーティリティ利用。
